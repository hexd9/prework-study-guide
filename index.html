  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.
  • The element that holds all the children elements is known as the Parent element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class attribute allows us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • A period preceding a class, designates it as a class selector

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it. Example: git checkout -b feature/add-html

JavaScript

  • A variable is a named container that allows us to store data in our code. Example: var shape = "circle";
  • In JavaScript, a string is a sequence of characters that is used to represent text. Strings can be written using either single quotes (' '), double quotes (" "), or backticks ( ) depending on the preference of the developer.
  • Control flow is the order in which a computer executes code in a script. Control flow can be interrupted, however. We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • Array is a data structure used to store a collection of values, such as numbers, strings, objects, or other arrays. Arrays are defined using square brackets [] and the values are separated by commas.
  • /* comments out code */
  • Function is simply a set of instructions that tells the computer how to perform a certain task.
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
